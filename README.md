# BreadKnife

A fully customizable desktop shell built with Rust, GTK4, and Astal — no compromises, no black boxes.

## Dependencies

### Arch Linux

```bash
sudo pacman -S rust libastal
```

Other distributions: build and install [Astal](https://github.com/aylur/astal) from source.

## Credits

- [Astal](https://github.com/aylur/astal) — Linux desktop shell framework
- [gtk4-rs](https://github.com/gtk-rs/gtk4-rs) — Rust bindings for GTK4

# BreadKnife

A fully customizable desktop shell built with Rust, GTK4, and Astal — no compromises, no black boxes.

## Dependencies

### Arch Linux

```bash
sudo pacman -S rustup
```

Install `libastal-meta` from **AUR**

### Other distributions

Build and install [Astal](https://github.com/aylur/astal) from source.

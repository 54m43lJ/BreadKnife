# BreadKnife

A fully customizable desktop shell built with Rust, GTK4, and Astal — no compromises, no black boxes.

## Build from Source

### Arch Linux

```bash
sudo pacman -S rustup gtk4 libadwaita meson desktop-file-utils gcc
```

Install `libastal-meta` from **AUR**.

Other distributions: build and install [Astal](https://github.com/aylur/astal) from source, then install GTK4 build dependencies for your distribution.
